## Allowlist download

You may download the allowlist [here](ips.txt?raw=1)

## Requirements

-   Rust >= 1.64.0

## Setup instructions

`git clone` and `cd` into the project directory, then run the following

```bash
cargo build
```

## Usage

```bash
cargo run
```
